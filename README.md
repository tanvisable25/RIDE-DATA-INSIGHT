# RIDE-DATA-INSIGHT

This project analyzes ride-booking data to understand overall booking performance, revenue trends, vehicle-type utilization, cancellation reasons, and customer/driver ratings. Interactive Power BI dashboards were created to provide insights into ride volume, payment behavior, cancellations, and service quality over a selected time period.

The ride-booking business needed a consolidated view of bookings, revenue, cancellations, vehicle performance, and customer satisfaction, as raw ride data made it difficult to track trends and operational inefficiencies.

To build an interactive Power BI dashboard that analyzes ride data to monitor performance metrics, identify revenue drivers, understand cancellation reasons, and evaluate customer and driver ratings.

- Cleaned and transformed raw ride data using **Power Query** to ensure data accuracy and consistency.
- Built a data model and created **DAX measures** for bookings, revenue, cancellation rate, and distance metrics.
- Designed multiple report pages covering **Overall performance, Vehicle Type, Revenue, Cancellation, and Ratings**.
- Implemented **date slicers, filters, and drill-downs** to enable dynamic analysis.
- Visualized insights using KPI cards, charts, and tables for easy interpretation by stakeholders.
- Delivered a centralized dashboard providing clear visibility into business performance.
- Helped identify high-performing vehicle types, key revenue sources, and major cancellation drivers.
- Enabled data-driven decisions to improve operational efficiency, reduce cancellations, and enhance customer experience.
